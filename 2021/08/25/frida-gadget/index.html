5.打包

直接apktool b xxx打包经常会报资源错误，可以尝试以下方式：

  • 直接删除出错的layout文件

  • 修改出错的layout文件

  • 不解包资源文件，也就是在解包的时候加入-r选项
    通过-f -p 选项使得打包时忽略这些资源
    apktool b -f -p . xxx -o output.apk

    -f => –force-all
    -p => resource path
